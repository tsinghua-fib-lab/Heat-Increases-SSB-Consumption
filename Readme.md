# Heat-Increases-SSB-Consumption

This repository contains the code and scripts used to reproduce the experimental results presented in our paper:  
**“Hot Weather Increases Sugar-Sweetened Beverage Consumption Through Online Food Delivery.”**

---

## Overview

All scripts required to replicate the analyses and figures in the paper are provided in this repository.

### Contents

- **`MakeFigure1.ipynb`**  
  Generates **Figure 1** as well as the results presented in **Supplementary Table S1** and **Supplementary Table S2**.

- **`MakeFigureS1.ipynb`**  
  Generates **Figure S1** and the results for the “City” category reported in **Table 1** of the main text.

- **`MakeFigureS2.ipynb`**  
  Generates **Figure S2** and the results for the remaining categories reported in **Table 1** of the main text.

- **`MakeFigureS3.ipynb`**  
  Generates **Figure S3**.

- **`MakeTableS3.ipynb`**  
  Generates the results presented in **Table S3**.

---

## Usage

Each Jupyter Notebook (`.ipynb`) can be run independently to reproduce the corresponding figure or table.  
To ensure reproducibility, please refer to the data and package requirements specified within each notebook.

Please note that, due to privacy restrictions, the original dataset used in this study cannot be publicly shared.
